## Hints

### Zsh

open zsh profile

```sh
code ~/.zshrc
```

```sh
echo "alias lg='lazygit'" >> ~/.bash_profile
```

reload zsh

```zsh
source ~/.zshrc
```

### Bash 

open bash profile

```sh
code ~/.bash_profile
```

add new alias

```sh
echo "alias lg='lazygit'" >> ~/.bash_profile
```

navigate to bash folder

```sh
cd  ~/.bash
```

reload bash

```sh
. ~/.bash_profile
```
